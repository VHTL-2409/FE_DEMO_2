Nhóm Khởi Tạo

__init__: tạo analyzer, load cascade/model, set threshold, tạo tracking state.
_env_bool: đọc biến môi trường kiểu boolean.
_env_float: đọc biến môi trường kiểu số thực.
_init_haar_cascades: load Haar cascade cho face, eye, smile, profile face.
_init_dnn_face_detection: load model DNN Caffe/TensorFlow để detect face.
_configure_dnn_net: ép DNN chạy backend OpenCV, target CPU.
_ensure_face_model_files: kiểm tra file model, thiếu thì tải.
_download_binary: tải file model từ URL về local.
_is_valid_file: kiểm tra file model có tồn tại và đủ size.
_init_eye_landmark_detector: khởi tạo MediaPipe FaceMesh hoặc OpenCV face module nếu có.
_init_spoofing_detector: load model anti-spoofing nếu có file weights.
Nhóm Trạng Thái Service

cv_ready: Haar/OpenCV cascade sẵn sàng chưa.
dnn_ready: DNN face model sẵn sàng chưa.
is_ready: có ít nhất DNN hoặc Haar dùng được chưa.
face_detector_status: trả trạng thái face detector cho diagnostics.
landmark_status: trả trạng thái landmark/FaceMesh.
eye_landmark_ready: landmark mắt dùng được chưa.
spoofing_dl_ready: model deepfake/spoofing DL dùng được chưa.
Hàm Chính

analyze_frame: lõi camera AI. Nhận image_base64, decode ảnh, detect face, detect mắt, gaze, spoofing, quality, sinh signal, trả JSON.
analyze_behavior: phân tích hành vi không từ camera: paste, tab switch, idle, typing anomaly.
Nhóm Decode + Response

_decode_image: base64 -> bytes -> PIL.Image RGB.
_signal: tạo object signal chuẩn: signal_type, severity, confidence, evidence.
_generate_warnings: từ signals tạo warnings text nếu có HIGH/CRITICAL.
Nhóm Camera/Mic Metadata

_resolve_camera_state: đọc cameraOn, trackEnabled, trackReadyState; nếu tắt thì NO_CAMERA.
_resolve_mic_state: đọc micOn, audioTrackEnabled, audioTrackReadyState; nếu tắt thì NO_MIC.
_build_camera_off_response: trả response riêng khi camera tắt, không cần phân tích ảnh.
_metadata_value: lấy value từ metadata theo nhiều alias.
_normalize_text: chuẩn hóa string.
_coerce_bool: đổi true/false string sang boolean.
_resolve_sample_time: lấy thời điểm frame từ capturedAt.
_to_millis: đổi timestamp sang milliseconds.
_resolve_tracking_key: lấy key theo attempt/session/student để tracking nhiều frame.
Nhóm Face Detection

_detect_face_locations: chọn DNN trước, Haar fallback sau.
_detect_faces_dnn: dùng OpenCV DNN Caffe detect mặt trên ảnh resize 300x300.
_detect_faces_haar: dùng Haar cascade detect mặt khi DNN không ổn.
_detect_with_cascade: wrapper gọi cascade.detectMultiScale.
_merge_face_locations: gộp box mặt bị detect trùng.
_rect_iou: tính overlap giữa 2 box.
_rect_center_close: kiểm tra 2 box có tâm gần nhau không.
_union_rect: gộp 2 box thành 1 box lớn.
Nhóm Eye Tracking

_track_eyes: hàm chính tìm mắt. Chỉ chạy khi có đúng 1 mặt.
_track_eyes_with_face_mesh: dùng MediaPipe FaceMesh nếu có để lấy landmark mắt/iris.
_landmark_point: đổi landmark normalized sang tọa độ pixel.
_detect_eye_candidates: dùng Haar eye cascade tìm vùng có thể là mắt trong nửa trên mặt.
_merge_eye_candidates: gộp box mắt bị trùng.
_select_eye_pair: chọn cặp mắt tốt nhất từ nhiều candidate.
_analyze_eye_roi: phân tích vùng mắt, tìm đồng tử tối, tính normalized_pupil, eye_state.
_resolve_eye_state: tổng hợp trạng thái mắt: OPEN, PARTIAL, CLOSED.
_resolve_eye_tracking_confidence: tính độ tin cậy tracking mắt.
Nhóm Gaze / Ánh Mắt

_analyze_gaze: từ vị trí đồng tử tính hướng nhìn CENTER/LEFT/RIGHT/UP/DOWN.
_update_tracking_state: giữ trạng thái qua nhiều frame, xác nhận nhìn lệch/nhắm mắt/mất mặt có ổn định không.
_ensure_tracking_state_defaults: đảm bảo các biến tracking tồn tại.
_prune_tracking_state: xóa tracking state cũ để tránh đầy memory.
Nhóm Quality Gate

_build_quality_gate: quyết định frame có đủ tốt để tin eye/gaze/position không. Check sáng, mờ, face size, obstruction.
_assess_face_quality: trả label chất lượng mặt: NO_FACE, MULTIPLE_FACES, EYES_NOT_VISIBLE, TOO_DARK, BLURRY, GOOD.
_assess_frame_quality: trả label chất lượng frame: POOR, FAIR, OVEREXPOSED, GOOD.
Nhóm Spoofing / Giả Mạo

_detect_spoofing_deep: điều phối các cách detect spoofing.
_dl_spoofing_detection: dùng model DL nếu có. Hiện repo thiếu weights nên thường không chạy.
_enhanced_statistical_spoofing: dùng thống kê ảnh, Laplacian, FFT.
_fft_screen_detection: tìm pattern màn hình/phát lại bằng FFT.
_texture_based_spoofing: kiểm tra texture mặt bằng Sobel edge.
_color_based_spoofing: kiểm tra entropy màu, nghi ảnh màn hình/ảnh giả.
Nhóm Occlusion + Position

_detect_occlusion: detect khẩu trang, kính râm, mặt bị che/một phần.
_analyze_face_position: xem mặt lệch tâm hoặc quay nghiêng.
_build_visual_overlay: tạo dữ liệu vẽ box mặt/mắt/pupil cho FE overlay.
_normalize_overlay_box: chuẩn hóa box overlay.
_normalize_overlay_point: chuẩn hóa point overlay.
Nhóm Signal Filter

_canonical_signal_type: chuẩn hóa alias signal về tên chuẩn, ví dụ AI_NO_MIC -> NO_MIC.
_signal_severity_rank: đổi severity thành số để sort.
_signal_priority: tính độ ưu tiên signal.
_filter_frame_signals: bỏ signal trùng, dedup theo thời gian, tránh spam.
