demo_2 — frontend song song với demo
======================================

Chạy dev (cổng mặc định 5173 — trùng với demo; chỉ chạy một trong hai nếu cùng máy):
  cd demo_2
  npm ci
  copy .env.example .env   (hoặc chỉnh VITE_API_BASE_URL → BE http://localhost:8082)
  npm run dev

Backend: CORS/WebSocket mặc định đã gồm http://localhost:5173.

Tham chiếu mockup HTML: thư mục ../stitch_new
Bảng map route: stitch-route-map.json

Preview: npm run preview (cổng 4173)
