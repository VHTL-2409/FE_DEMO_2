Reverse proxy — Nginx trên VPS → Docker (FE_DEMO)
==================================================

Luồng chung
-----------
  Nginx HOST  →  proxy_pass / → 127.0.0.1:8080 (fe), /api /ws → 127.0.0.1:8082 (be)

  Docker: DOCKER_FE_PUBLISH=127.0.0.1:8080 và DOCKER_BE_PUBLISH=127.0.0.1:8082

  Build lại fe khi đổi VITE_*:
  docker compose -f docker-compose.prod.yml --env-file .env build --no-cache fe
  docker compose -f docker-compose.prod.yml --env-file .env up -d


Giới hạn (trình duyệt)
----------------------
  - URL "không ghi cổng" + HTTPS → mặc định là cổng 443. Không dùng 443 thì không có
    https://domain (không cổng) chuẩn — trừ khi dùng cổng lạ và GHI rõ (vd. :444).
  - URL "không ghi cổng" + HTTP → mặc định cổng 80 → http://domain


0) HTTP cổng 80 — không ghi :80, KHÔNG dùng 443, KHÔNG dùng :444
-----------------------------------------------------------------
  Truy cập: http://test1.nexquoteapp.com

  Mẫu: deploy/nginx-host-80-http-only.conf.example
  Firewall: sudo ufw allow 80/tcp

  .env (http, không cổng trong URL):

  APP_FRONTEND_BASE_URL=http://test1.nexquoteapp.com
  APP_CORS_ALLOWED_ORIGINS=http://test1.nexquoteapp.com
  VITE_API_BASE_URL=http://test1.nexquoteapp.com

  Lưu ý: không mã hóa TLS — chỉ phù hợp mạng nội bộ / demo. Cần HTTPS mà không dùng 443
  → phải ghi cổng (vd. https://domain:444) hoặc dùng mục 1.


1) HTTPS cổng 443 — không ghi :443 trên URL (https://domain)
-------------------------------------------------------------
  Truy cập: https://test1.nexquoteapp.com

  Mẫu: deploy/nginx-host-80-443.conf.example
  Firewall: 80, 443
  .env: https://domain (không cổng)
  SSL: certbot --nginx -d domain


2) HTTPS cổng 444 — khi 80/443 đã bận
--------------------------------------
  Truy cập: https://test1.nexquoteapp.com:444  (bắt buộc ghi :444)

  Mẫu: deploy/nginx-host-81-444.conf.example
  Firewall: 81, 444


Kiểm tra
--------
  HTTP:  curl -I http://test1.nexquoteapp.com/
  HTTPS: curl -I https://test1.nexquoteapp.com/


Spring Boot
-----------
  server.forward-headers-strategy=framework (application.properties)
